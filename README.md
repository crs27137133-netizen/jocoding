# 로또 번호 생성기

1-45 사이의 랜덤 로또 번호를 추천해주는 웹사이트입니다.

## 기능

- 🎲 1-45 사이의 중복 없는 랜덤 번호 6개 자동 생성
- 🌈 번호마다 다른 색상의 볼 디자인
- 📝 최근 5개까지 생성 기록 저장
- 🧹 기록 삭제 기능
- 📱 반응형 디자인 (모바일에서도 잘 보임)
- ✨ 부드러운 애니메이션 효과

## 사용 방법

`index.html` 파일을 브라우저로 열면 바로 사용할 수 있습니다.
